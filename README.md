# Ebon Might Extension Helper

- [Curseforge]
- [Wago]

## Features

Plays a sound when you start casting an extender (Eruption, Upheaval, Fire Breath) and:

- you have no Ebon Might active
- the cast would end after expiration of the aura

In either case, you'd be wasting resources.

You can customize the sound played and the channel it's played on through the settings menu and also optionally disable the addon.

The addon will also exclusively run on Evokers and be effectively inactive until you spec into Augmentation.
